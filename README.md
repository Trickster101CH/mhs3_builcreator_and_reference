# MHS3 Build Creator + Counter

Two browser-based tools for **Monster Hunter Stories 3**.

---

## Build Creator

**Path:** `build_creator/mhs3_build_creator.html`

An HTML tool for creating and planning builds in MHS3. Uses a local JSON file (`mhs3_skills.json`) as a data source for available skills.

### Usage

- Open `mhs3_build_creator.html` in your browser
- Select monsters and skills to put together builds

---

## Counter

**Path:** `counter/mhs3_counter.html`

An HTML tool to display enemy monster attack patterns including weaknesses.

### Usage

- Open `mhs3_counter.html` in your browser

---

## Requirements

- A modern web browser (Chrome, Firefox, Edge, etc.)
- No installation or server required – simply open the HTML files locally
